livius
======

livius - a Qt-based TLCV-compatible live chess viewer

build instructions
------------------

$ make

Requires Qt 5, in particular:
- qmake
- qtdeclarative5-dev
- qtdeclarative5-dev-tools
- libqt5svg5-dev
